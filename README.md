 # 🔐 Password Generator

Удобный консольный генератор безопасных паролей на *Python*.

## 📖 Описание
Этот проект предназначен для быстрого создания надежных паролей прямо в терминале. Программа генерирует случайную комбинацию символов, которую сложно взломать методом перебора (brute-force).

## 🚀 Как запустить проект

1. *Клонируйте репозиторий:*
   ```bash
   git clone https://github.com/Regina493/password-generator.git
   """
## 🛠 Технологии
* *Python 3.x* — основной язык разработки.
* *модуль random* — для генерации случайных последовательностей.
* *модуль string* — для работы с наборами символов (буквы, цифры, знаки).

## ✨ Функционал
- [x] Генерация пароля в один клик.
- [x] Использование символов верхнего и нижнего регистра.
- [x] Включение цифр и специальных знаков.
- [x] Высокая скорость работы.
